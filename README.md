# Particles-JavaScript-
### **Использование**
-------------------------

Скачайте particles.js и поместите его в свой проект:

**index.html** 
```html
 <script src="particles.js"></script>
```
Подключить конфигурационный файл
```html
 <script src="config.js"></script>
```
